First unzip the downloaded file and follow below steps

Method 1: 
    1.	First Login to admin panel 
    2.	Go to : (Preference) > (Configuration) 
        Set, Debug Mode = 1
    3.	Click on (Application) tab
    4.	Click on (Install New Component)
    5.	Select contactus.zip file ( Click on the Floppy Disk image on right side of the upload file input box)
    6.	Click on Install Button. 
    7.	After upload, this Contactus component will appear in the list.
        Now click on "Activate" link blow description. 
        
Method 2: 
    1.	First Login to admin panel 
    2.	Go to : (Preference) > (Configuration) 
        Set, Debug Mode = 1
    3.	Unzip contactus.zip.
    4.	Copy "contactus" folder in "component" directory in your project
    5.	In Admin panel go to application tab and find contactus component
    6.	Click on Activate Link.
